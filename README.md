# 💰 Spendwise — Expense Tracker

A full-stack expense tracking web app built with **Vanilla PHP**, **SQLite**, **CSS**, and **JavaScript**.

---

## 🚀 Features

- ✅ Add **income & expenses** with categories
- ✅ **Category filter** (Food, Bills, Transport, etc.)
- ✅ **Monthly filter** — view any month
- ✅ **Balance summary** (Income, Expenses, Net Balance)
- ✅ **Pie chart** (Chart.js) — expense breakdown by category
- ✅ **Category breakdown** with progress bars
- ✅ **Delete** transactions with smooth animation
- ✅ **Animated counters** on the stat cards
- ✅ **Toast notifications** for feedback
- ✅ **Zero-config** — uses SQLite, no MySQL setup needed
- ✅ Demo data pre-loaded on first run

---

## 🛠️ Tech Stack

| Layer     | Tech                        |
|-----------|-----------------------------|
| Backend   | PHP 8+ (PDO + SQLite)       |
| Frontend  | Vanilla HTML/CSS/JavaScript |
| Charts    | Chart.js 4                  |
| Database  | SQLite (auto-created)       |

---

## ⚙️ Setup (Local)

### Requirements
- PHP 8.0+
- PHP SQLite extension enabled (`php_pdo_sqlite`)

### Run locally
```bash
cd expense-tracker
php -S localhost:8000
```
Then open: **http://localhost:8000**
